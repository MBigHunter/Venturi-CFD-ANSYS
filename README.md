Project Overview

This project presents a numerical simulation of incompressible flow through a Venturi tube using ANSYS Fluent.
The objective was to analyze velocity distribution, pressure drop, and validate the numerical results with analytical predictions.

Model Description

Flow type: Laminar

Reynolds number(max): 4.2

Inlet velocity: 0.01

Fluid: Glycerin

Mesh type: Structured

Number of elements: 20634

Mesh Independence Study

A mesh independence study was performed to ensure the accuracy of numerical results.

Validation

Simulation results were compared with theoretical predictions based on Bernoulli equation.

Conclusion

The CFD results showed good agreement with analytical solutions and demonstrated the effectiveness of numerical methods in analyzing Venturi flow behavior.
